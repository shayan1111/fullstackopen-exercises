# Phonebook App

This is a simple Phonebook application built with React and Node.js (Express).  
At this stage of the course, you can **only view the contacts** in the phonebook.

## Deployed Application

You can access the live app here: https://fullstackopen-exercises-1-c6ab.onrender.com/

## Features

- View all contacts

## Technologies Used

- Frontend: React, JavaScript
- Backend: Node.js, Express
- Deployment: Render
